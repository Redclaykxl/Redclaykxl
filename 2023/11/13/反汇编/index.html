Download Python | Python.org这是python的官网,从这里现在最先版本的python.
运行安装程序，确保选择了“Add Python x.x to PATH”选项，这会自动将Python添加到系统环境变量。

手动添加Python和pip到环境变量（如果未自动添加）:
  • 右键点击“此电脑”或“我的电脑”，选择“属性”。
  • 点击“高级系统设置”。
  • 在弹出窗口中，点击“环境变量”按钮。
  • 在“系统变量”部分，找到名为“Path”的变量，点击“编辑”。
  • 点击“新建”，然后添加Python和pip的安装路径（通常是 C:\Users\YourUsername\AppData\Local\Programs\Python\PythonXX 和 C:\Users\YourUsername\AppData\Local\Programs\Python\PythonXX\Scripts）。
  • 确认并关闭所有窗口。
